


# 阿熊的抽獎小網
### github page:  https://reonabear.github.io/twitch-Raffle/
可以讓使用者  
- 轉換excel或試算表名單，將其依數字排列，好讓抽獎進行
- 複製抽獎名單
- 依照不同的獎項進行過濾
- 刪除抽中的那筆資料
- 複製中獎者名單，並自動扣除編號

(格式為 名字 數字)
